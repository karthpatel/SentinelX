# SentinelX

SentinelX is a dark cybersecurity web app for everyday security checks.

## Features

- Cyber Hygiene Assessment
- URL Threat Analyzer
- Scam Message Analyzer
- Dashboard with recent activity and risk statistics
- Browser-based history using local storage

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
